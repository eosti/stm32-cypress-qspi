Dummy cycle configuration

Dummy cycles for SDR, High Performance

Precondition
Define QSPI_DUMMY_{50 | 80 | 90 |104} based on the QSPI peripheral clock speed
Return values
DUMMY_CLOCK_CYCLES_READdummy clock cycles for READ reads
DUMMY_CLOCK_CYCLES_FASTREADdummy clock cycles for FASTREAD reads
DUMMY_CLOCK_CYCLES_READ_DUALdummy clock cycles for DUAL reads
DUMMY_CLOCK_CYCLES_READ_DUALIOdummy clock cycles for DUALIO reads
DUMMY_CLOCK_CYCLES_READ_QUADdummy clock cycles for QUAD reads
DUMMY_CLOCK_CYCLES_READ_QUADIOdummy clock cycles for QUADIO reads
DUMMY_LCLatency code for given speed
Postcondition
User must set DUMMY_LC in CR1 for any read operations
